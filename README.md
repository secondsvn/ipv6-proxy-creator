## Requirements
- Centos 7
- Ipv6 \64

## Installation
1. `bash <(curl -s "https://raw.githubusercontent.com/secondsvn/ipv6-proxy-creator/main/scripts/install.sh")`
   
2. After installation dowload the file `proxy.zip`
   * File structure: `IP4:PORT`
   
## Test your proxy
Open [ipv6-test.com](http://ipv6-test.com/) and check your connection
